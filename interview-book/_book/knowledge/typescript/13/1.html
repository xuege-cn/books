lib.d.ts

lib.d.ts

安装 Typescript 时，会顺带安装一个lib.d.ts，tsconfig.json中有noLib可以排除此文件

const foo = 123;
foo.toString();

Window，Document，Math，String

可以对这些声明进行拓展，建议在 global.d.ts 中
interface Window {
    helloWorld(): void
}

interface Math {
  seedrandom(seed?: string): void;
}

interface String {
  endsWith(suffix: string): boolean;
}
