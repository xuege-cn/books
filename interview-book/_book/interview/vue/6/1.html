vue如何做编译时优化AOT？

JIT

即时编译：just-in-time compile

React 的模版中允许写 js 代码，然而 vue 的模版中不允许写 js 代码

  1. React 是即时编译，会在执行时把 jsx 编译为 js 代码
  2. Vue 是即时编译兼运行前编译，就看 $options.render 是否存在，存在为运行前编译，反之即时编译

AOT

运行前编译：Ahead-of-time compile
