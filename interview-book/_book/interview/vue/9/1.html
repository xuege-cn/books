vue-router 使用

动态路由匹配

  1. /user/:id 匹配 /user/foo, /user/bar, this.$router.params.id 获取参数
响应路由参数变化

从 /user/foo 到 /user/bar，User 组件会被复用，因为是同一组件，复用比销毁重建更高效，但是组件的生命周期钩子不会再被调用
如果想对路由参数变化作出响应，如下👇：

方案一：
watch: {
    $route (to, from) {
        // 监听路由参数变化
    }
}

方案二：导航守卫
beforeRouteUpdate (to, from, next) {
    next()
}
